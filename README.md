
  # Quiki Mobile App Development

  Available to view at https://piano-gear-16314438.figma.site/

## Mobile App Prototype Overview

This app is a unified grocery-delivery aggregator designed to help users compare prices, delivery windows, fees, and product availability across platforms like Uber Eats, Instacart, Amazon Fresh, Walmart+, Shipt, and others — all in one place.
The prototype showcases how users can seamlessly browse, optimize, and order groceries from multiple providers through a single, intuitive interface.

## Goals of the Prototype

 - Validate core UX flows before engineering investment
 - Gather actionable user feedback through realistic interactions
 - Demonstrate primary use cases, screens, and micro-interactions
 - Align cross-functional teams around a shared product vision
 - Lay the foundation for multi-platform comparison and intelligent cart optimization

## What’s Included

 - Fully clickable high-fidelity screens
 - Core flows: onboarding, multi-store comparison, cart optimization, checkout, and settings
 - Interaction logic that mimics final app behavior
 - Component library for consistent UI elements
 - Notes on user insights and design decisions

## Tech & Tools Used

 - Figma (UI design + prototyping)
 - iOS/Android design systems
 - User research insights and journey mapping
